# 03 · 冷却：最近 3 家已接受决策排除出候选集

## Parent

- [PRD: 今天吃什么 MVP](../PRD.md)

## What to build

转盘抽之前，决策引擎先读取决策存储，把最近 3 条已接受决策对应的餐厅从候选集中排除（**冷却**）。冷却全自动，用户无感。效果：连续转盘不会反复给出刚选过的店。冷却是系统自动态，区别于用户当次设置的**约束**。

## Acceptance criteria

- [ ] 转盘前引擎读取决策存储，将最近 3 条已接受决策的餐厅排除出候选集
- [ ] 连续转盘不会给出最近 3 家已接受的餐厅
- [ ] 冷却全自动，无需用户操作
- [ ] 引擎单测覆盖：给定冷却列表，候选集正确排除对应餐厅
- [ ] 冷却窗口为 3（可调，但默认 3）
