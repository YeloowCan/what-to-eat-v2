# 05 · 金色锁定接受

## Parent

- [PRD: 今天吃什么 - 界面重做](../PRD.md)

## What to build

把「就这家」做成金色「锁定」--命运认定的情绪收尾(ADR-0004)。接受时:金色光晕从卡片外荡 + 卡头盖「就这家✓」金色小戳 + 卡片 scale 收束 + 底部按钮形变成金色「去美团看看」深链。金色仅用于此处与转盘落定闪(见 ADR-0004),不另引入他处。

## Acceptance criteria

- [ ] 点「就这家」触发金色光晕从卡片外荡
- [ ] 卡头盖「就这家✓」金色小戳
- [ ] 卡片做轻微 scale 收束
- [ ] 底部按钮形变成金色「去美团看看」
- [ ] 金色仅出现在此处与转盘落定闪(不引入其他金色)
- [ ] 接受后跳美团深链仍可用;失败仍有兜底提示
- [ ] 接受仍落决策 + 更新冷却(沿用既有行为)
- [ ] 微信开发者工具验证通过
