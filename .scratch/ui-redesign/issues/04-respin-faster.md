# 04 · 重抽重转(提速)

## Parent

- [PRD: 今天吃什么 - 界面重做](../PRD.md)

## What to build

让「换一家」重跑真转盘(而非静默刷卡片),使每次重抽都是又一次命运抽签--但比首抽更快,连抽不拖。衔接:卡片出 -> 转盘回 -> 快转 -> 落定 -> 卡片涌。

## Acceptance criteria

- [ ] 「换一家」重跑转盘动画(不是静默刷新卡片)
- [ ] 重抽时长短于首抽(~1.2s vs ~2.5s)
- [ ] 卡片出->转盘回->快转->落定->卡片涌 衔接顺滑、无闪烁
- [ ] 重抽仍不落决策、不进冷却(重抽是临时展示,沿用既有行为)
- [ ] 微信开发者工具验证通过

## Blocked by
