# 06 · 约束摘要 chip + 历史图标 + 各状态调色

## Parent

- [PRD: 今天吃什么 - 界面重做](../PRD.md)

## What to build

让转盘唱主角:约束控件收成顶部一颗可点摘要 chip(如「1km · 不限 · 营业中」),点开才展开完整选择器;「最近选过」从底部按钮收成右上角小图标。把所有过渡/错误状态(定位中 / 定位被拒 / 定位失败 / 网络错 / needsRelaxCuisine / 空)纳入命运调色(暖而不刺眼),定位中用轻量金色动效替掉灰字。历史页同步套调色。

## Acceptance criteria

- [ ] 约束显示为顶部单颗摘要 chip;点击展开完整选择器(距离 / 菜系 / 营业中)
- [ ] 改约束仍自动重转
- [ ] 「最近选过」为右上角小图标(移除底部按钮);点击进历史
- [ ] 定位中状态用轻量金色动效(无裸灰字)
- [ ] 定位被拒 / 定位失败 / 网络错 / needsRelaxCuisine / 空 各状态均套命运调色,暖而不刺眼
- [ ] 历史页套命运调色;快照行为不变(店关改名仍显示当时选的)
- [ ] 约束 chip + 历史图标不挤占转盘主舞台
- [ ] 微信开发者工具验证通过
